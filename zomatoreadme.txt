# 🍽️ Zomato NLP Cuisine Recommender Web App

This is a Streamlit-based web application that uses NLP and Machine Learning to cluster Zomato cuisines and analyze customer reviews.

## 🔍 Features
- 📊 Cuisine clustering using KMeans
- 🌥️ Word Cloud generation from review text
- 😊 Sentiment analysis with TextBlob
- 🔎 Search restaurants by cuisine or cluster
- 📍 Filter by location
- 📥 Export sentiment analysis results
- 🧠 Built with Streamlit for fast deployment

## 🧪 Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib
- WordCloud
- TextBlob
